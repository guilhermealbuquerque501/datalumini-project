<h1>Projeto-PI</h1>

<h2>Coleta e Análise de Dados de Incidência Luminosa na Planta <i>Arabidopsis thaliana</i> em Ambientes Laboratoriais</h2>

<h3>Sobre o Projeto</h3>
<p>
Este projeto tem como objetivo coletar e analisar dados relacionados à incidência luminosa na planta 
<i>Arabidopsis thaliana</i> em ambientes laboratoriais controlados.
</p>

<p>
A proposta é simular e avaliar os possíveis ganhos biológicos e energéticos ao utilizar um sistema 
inteligente de monitoramento da iluminação. Com isso, busca-se identificar o ponto ideal de luminosidade 
para o crescimento da planta, evitando desperdício de energia e reduzindo o estresse térmico causado pelo excesso de luz.
</p>

<h3>Objetivos</h3>
<ul>
<li>Monitorar a incidência luminosa em plantas cultivadas em laboratório.</li>
<li>Analisar dados de luminosidade e resposta da planta.</li>
<li>Simular ganhos de produtividade e eficiência energética.</li>
<li>Reduzir desperdício de energia causado por iluminação excessiva.</li>
</ul>

<h3>Planta Utilizada</h3>
<p>
A planta utilizada no estudo é a <i>Arabidopsis thaliana</i>, amplamente utilizada em pesquisas científicas por possuir:
</p>

<ul>
<li>Ciclo de vida curto</li>
<li>Genoma bem mapeado</li>
<li>Facilidade de cultivo em laboratório</li>
</ul>

<h3>Tecnologias Utilizadas</h3>
<ul>
<li>Sensores de luminosidade</li>
<li>LEDs para controle de iluminação</li>
<li>Banco de dados para armazenamento dos dados</li>
<li>Linguagens de programação para análise e visualização dos dados</li>
</ul>

<h3>Funcionamento do Projeto</h3>
<ol>
<li>Sensores coletam dados da incidência luminosa.</li>
<li>Os dados são enviados para um sistema de armazenamento.</li>
<li>O sistema analisa os valores coletados.</li>
<li>São feitas simulações para avaliar eficiência energética, crescimento da planta e possíveis economias no uso de iluminação.</li>
</ol>

<h3>Resultados Esperados</h3>
<ul>
<li>Otimização do uso de iluminação em ambientes de cultivo.</li>
<li>Redução do consumo energético.</li>
<li>Melhor controle do crescimento da planta em laboratório.</li>
</ul>
